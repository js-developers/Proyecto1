Estos son los comandos relevante para la utilizacion de git, para manejo de versionamiento de archivos


  1. git init "ESte comando se utiliza para iniciar o crear el repositorio en local, si no se creael repositorio no sera posible agregar los archivos"
  2. ls -la "Este comando nos permite ver los archivos que el repositorio"
  3. git add name file "Este nos permite agregar archivos a nuestro repositorio local"
  4. git status "Este comando sirve para ver el estado de nuestros archivos, debido a que este programa trabaja con varias fases para nuestros archivos, al momento de ingresar un archivo con el git add podemos encontrarlo en una seccion llamada index"
  5. git commit -m "mensaje" "Este comando sirve para enviar por completo nuestro archivo al repositorio local"
  6. git config --global -l "Este comando lo que hace es visualizar las credenciales registradas en nuestro git."
  7. git log "Este comando cumple la funcion de mostrar el registro de todos los commit registradops y el usuario que los realizo"
  8. git log --oneline "Este comando muestra un informe de commit un poco menos detallado y en una sola linea"
  9. git remote add "nombre o alias para el remoto" "Direccion del repositorio creado online" "Este comando conecta nuestro repositorio local con el web generado por githud "
  10. git remote -v "Este comando sirve para listar los repositorios remotos"
  11. git push "alias del remoto agregado" rama del local "maste" "Este comando sirve para subir nuestro proyecto local al servidor, cuando nos referimos a la rama en este caso master equivale a la ubicacion donde tenemos nuestro proyecto en local"
  12. git clone [url del repositorio creado en github] "Este comando nos permite utilizar un repositorio que hayamos creado inicialmente en github y clonarlo a nuestra maquina, para trabajarlo de forma local
  13. git rename "alias del repositorio" "Este comando nos permite renombrar o cambiar el alias asignado anteriormente
  14. git reset --hart "Este comando nos permite regresar a la version anterior de nuestro commit
  15. git rm nombre del archivo "Este comando nos permitira remover archivos ya guardados anteriormente en el repositorio
