Estos son los comandos relevante para la utilizacion de git, para manejo de versionamiento de archivos


  • El primer comando es git add name file "Este nos permite agregar archivos a nuestro repositorio local"
  • El segundo comando es git status "Este comando sirve para ver el estado de nuestros archivos, debido a que este programa trabaja con varias fases para nuestros archivos, al momento de ingresar un archivo con el git add podemos encontrarlo en una seccion llamada index"
  • El tercer comando es git commit -m "mensaje" "Este comando sirve para enviar por completo nuestro archivo al repositorio local"
  • El terver comando es git config --global -l "Este comando lo que hace es visualizar las credenciales registradas en nuestro git."
  • El cuarto comando es git log "Este comando cumple la funcion de mostrar el registro de todos los commit registradops y el usuario que los realizo"
  • El quinto comando es git log oneline "Este comando muestra un informe de commit un poco menos detallado y en una sola linea"
  • Sexto comando git remote add "nombre o alias para el remoto" "Direccion del repositorio creado online" "Este comando conecta nuestro repositorio local con el web generado por githud "
  • Septimo comando git remote -v "Este comando sirve para listar los repositorios remotos"
  • Octavo comando git push "alias del remoto agregado" rama del local "maste" "Este comando sirve para subir nuestro proyecto local al servidor, cuando nos referimos a la rama en este caso master equivale a la ubicacion donde tenemos nuestro proyecto en local"
